# Quiz App

A simple and interactive Quiz Application built using HTML, CSS, and JavaScript. Users can answer multiple-choice questions, view their final score, and restart the quiz at any time.

## 🚀 Features

* Multiple-choice questions
* One question displayed at a time
* Next and Submit buttons
* Real-time score tracking
* Final score display
* Restart quiz functionality
* Clean and responsive design
* Smooth transitions between questions

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🎯 How It Works

1. Start the quiz.
2. Select an answer for each question.
3. Click **Next** to move to the next question.
4. Click **Submit** on the final question.
5. View your final score.
6. Click **Restart** to play again.
